# Streamlit Community Cloud Deployment Guide

1. Push the extracted `COS40007-Project` folder contents to GitHub.
2. Confirm `app.py` is at repository root.
3. Confirm `requirements.txt` is at repository root.
4. Go to Streamlit Community Cloud.
5. Create a new app from the GitHub repo.
6. Set main file path to `app.py`.
7. Deploy.

Do not use Google Colab paths or Windows backslash paths in the Streamlit app.
