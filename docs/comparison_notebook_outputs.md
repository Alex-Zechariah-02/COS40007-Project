# Official Comparison Notebook Outputs

The comparison section uses the saved outputs from `COS40007_SVR_RF_Comparison_Demonstrator` as the source of truth.

## Output groups

- `data/comparison/checks/` contains input availability, schema validation, row alignment, actual-target consistency, held-out year, forecast preview, and output completeness checks.
- `data/comparison/tables/` contains official comparison tables for metrics, baselines, row-level prediction comparison, grouped errors, feature influence, AI demonstrator, forecast preview, and final model selection.
- `figures/comparison/` contains official visual evidence generated by the comparison notebook.
- `data/comparison/artifacts/` contains the comparison output manifest and final supervised model selection artifact.

## Current selected model

Support Vector Regression is selected as the stronger tested supervised-regression model. It outperforms RandomForestRegressor on MAE, RMSE, R², Median AE, row-level wins, tolerance coverage, and most grouped-error comparisons.

## Interpretation limit

Both models have negative held-out R², so the comparison should be interpreted as a model-selection result for a planning-support prototype, not as evidence of production-grade forecasting accuracy.
